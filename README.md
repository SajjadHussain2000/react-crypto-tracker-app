
# **Cryptocurrency Tracker**

A sleek and interactive cryptocurrency tracking application providing real-time data on cryptocurrency trends and market metrics.

---

## **Features**
1. **Real-Time Market Updates**:
   - Fetches live cryptocurrency data using the CoinGecko API.

2. **Dynamic Charting**:
   - Visualize trends with **Chart.js**.

3. **Responsive Design**:
   - Seamlessly adapts to all devices using Material-UI.

---

## **Tech Stack**
- **Frontend**: React.js, Material-UI  
- **Charting**: Chart.js  
- **API Integration**: CoinGecko API  

---

## **Setup Instructions**

### **1. Clone the Repository**
```bash
git clone https://github.com/SajjadHussain2000/react-crypto-tracker-app.git
cd react-crypto-tracker-app
```

### **2. Install Dependencies**
```bash
npm install
```

### **3. Run the Application**
```bash
npm start
```
The app runs on `localhost:3000`.
